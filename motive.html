Motivation

To reduce the time required to compare the images in a dataset and retrieve the Close the matches with little or no effect to the accuracy of results

  • Linear Comparison with N dataset Images
    • Linear comparison of images involving N images takes N comparisons and each comparation involving calculating the difference of the 128 feature set and calculating the L2 Norm of the difference
    • Disadvantages with this approach
      1. With the Use case of needing the Top 10 results, we unnecessarily compare faces even though two faces are distinct from each other
      2. Additionally we Compare all the 128 features there might be some components in the matrices that do no affect the results.
  • K-Means Cluster Comparison
    • Since K-Means is a pre-processing step, We compare the input face only with images in the closely matching cluster therby reducing the N value
  • Principal Components Comparison
      As a pre-processing step , we calculate the principal components of the dataset and use only those components for comparison say we reduce the matrix from 128 to 7 features set . So Now we run the comparing step on these reduced principal feature set.

The above techniques makes the computation of matching results quicker even though there might be slight negligible trade-off with accuracy .This kind of techniques will be useful in automatic face detection and face tagging techniques available in Social Networks.
