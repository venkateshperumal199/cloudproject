Overview

Proposed Project Goal

The project is to develop a language model using three different classification algorithms which classifies the input corpus based on the authorâ€™s name, and then evaluates them based on the accuracy of classification

Algorithms

  • Naive Bayes Classifier
    • It is a classification technique based on Bayes' Theorem with an assumption of independence among predictors. In simple terms, a Naive Bayes classifier assumes that the presence of a particular feature in a class is unrelated to the presence of any other feature
  • Logistic Regression
    • We would be providing classification results for each author and approach the problem as a 2-class classifier and not a k-class classifier. The result would hence be of the form: Document belongs to the author or Document does not belong to the author. We then bifurcate the data into a training set (70%) and a test set (30%). We train the logistic regression model on the training set and predict the outcome based on the test set.
  • K Nearest Neigbors
    • Computing the TFIDF scores of all unique words is computed for each Author in training corpus. TFIDF scores of all unique words is computed for each author in test corpus.Euclidean distance is calculated by comparing the selected test document TFIDF scores with TFIDF scores of each document present in the training set. The documents are ranked in ascending order. K value is selected, and first K number of documents are selected from the ranked list. Author with highest number of document in the K list, classified as author for the test document selected.
