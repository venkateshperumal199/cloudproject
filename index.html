Overview

Proposed Project Goal

To compare the performance , computational complexity and accuracy of results using K-means and PCA on matching similar faces from a large dataset of faces of individuals using 128 distinct measurements from every face.

Stages

  • Linear Comparison
    • Involves Linear Comparison (i.e Comparison of input image matrix with matrix of all the images in the dataset.Calculating the norm of difference of the two matrices- Face Distance and returning the top 10 with least distance) of Face encodings ( a (1,128) image feature matrix ) generated using the Face Recognition library
  • Pre-processed K-Means Cluster based Comparison
    • Running K-Means on the image feature set of entire dataset and forming Clusters as a Pre-processing Step.
    • Runnning the Comparation on the pre-processed clusters and identifying the top 3 Clusters with least Face distance to the input image and retrieving the top 10 images with least face distance in those Clusters
  • Pre-processed Principal Components based comparison using Principal Component Analysis as a pre-processing step
    • Running Principal Component Analysis on the image feature set of entire dataset and forming a a matrix with M principal features instead of D features as a Pre-processing Step.
    • Computing the matrix of M feature set for input image using the mean and principal components of dataset (stored in previous step) and comparing it with dataset images with M features stored as a intermediate file in the previous step .
